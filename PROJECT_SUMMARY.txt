═══════════════════════════════════════════════════════════════════════════════

НАЗВА ПРОЕКТУ:
  Telegram Taxi Service Web App - платформа замовлення послуг через Telegram

ОСНОВНА ФУНКЦІОНАЛЬНІСТЬ:
  ✅ Замовлення таксі, вантажних перевезень, кур'єрських послуг, евакуатора
  ✅ Система ставок - водії пропонують ціни, клієнти приймають/відхиляють
  ✅ Реал-тайм чат між клієнтом та водієм
  ✅ Профіли водіїв з рейтингом (попередження/бонуси)
  ✅ Адміністративна панель для управління водіями
  ✅ Система кодів доступу для реєстрації водіїв
  ✅ Повна Telegram Bot інтеграція з webhook

РОЛІ:
  1. Клієнт     - замовляє, чекає пропозицій, обирає водія
  2. Водій      - бачить замовлення, пропонує ціни, виконує замовлення
  3. Адміністратор - управління водіями, видача кодів, модерація

ТЕХНІЧНИЙ СТЕК:
  Frontend:   React 18 + TypeScript + Vite + Shadcn/UI + Tailwind CSS + Wouter
  Backend:    Express.js + Node.js ESM
  State:      React Context + TanStack Query v5
  Forms:      React Hook Form + Zod
  Database:   PostgreSQL + Drizzle ORM
  Bot:        Telegram Bot API + Webhook

ФАЙЛИ ДЛЯ КОПІЮВАННЯ (Обов'язкові):
  ✓ server/storage.ts              - CRUD операції, весь функціонал
  ✓ server/routes.ts               - Всі REST API endpoints
  ✓ server/telegram.ts             - Телеграм бот, webhook, команди
  ✓ shared/schema.ts               - Описи таблиць, Zod схеми
  ✓ client/src/pages/              - 9 сторінок (роутинг, формі, спискі)
  ✓ client/src/lib/                - UserContext, QueryClient, hooks
  ✓ client/src/components/         - Shadcn компоненти + chat.tsx
  ✓ client/src/App.tsx             - Router з 8 маршрутами
  ✓ client/src/index.css           - Дизайн система, кольорові змінні

НАЛАШТУВАННЯ:
  1. TELEGRAM_BOT_TOKEN - встановіть на Replit Secrets
  2. Admin Password - admin123 (змініть для продакшену)
  3. DATABASE_URL - автоматично на Replit, інакше встановіть своєю
  4. APP_URL - встановлюється з REPL_ID/REPL_OWNER автоматично

ШВИДКИЙ СТАРТ:
  1. Скопіюйте весь проект на Replit
  2. npm install
  3. npm run dev
  4. Напишіть своєму Telegram боту /start
  5. Вибирайте роль та тестуйте

РЕЗУЛЬТАТ:
  - Полнофункціональна платформа замовлення послуг
  - Мобільно-оптимізована для Telegram
  - Готова до розширення та модифікації
  - Без залежностей від третіх сервісів (окрім Telegram)
